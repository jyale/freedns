Welcome to the FreeDNS project page.

Project Overview

The FreeDNS project is a Yale University computer science collaboration between Dr Ramki Gummadi, Prof Bryan Ford, Prof Avi Silberschatz and John Maheswaran.

Firefox Extension [download]

To access FreeDNS you need to download the FreeDNS extension for Mozilla Firefox. To install it just drag and drop the downloaded file (freedns.xpi) into Firefox and it will ask if you want to install the FreeDNS extension. Click install and you will be prompted to restart Firefox after which you will be able to use the plugin to access FreeDNS.

Once you've installed the plugin, you will see a menu item in Firefox that allows you to turn on and turn off access to FreeDNS (see picture below). Clicking "Toggle FreeDNS access" will turn access to FreeDNS on and off.



Once installed, you will be able to access the FreeDNS world, that is addresses that end in the suffix ".freedns" such as example.freedns. Simply navigate to that page in your web browser. (Note that FreeDNS is an ongoing project still under development and all .freedns addresses currently redirect to dedis.cs.yale.edu.)
